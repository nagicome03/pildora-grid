# 💊 Píldora GRID: Conceptos de Teoría

Este repositorio contiene los ejemplos prácticos desarrollados para una **Píldora GRID**. 

## 🛠️ Tecnologías Utilizadas
Para la construcción de estos ejemplos y el despliegue del material, se han utilizado:

* **HTML5:** Estructura semántica de los ejemplos.
* **CSS3:** Estilos y maquetación visual.
* **Git:** Control de versiones para el seguimiento del desarrollo.
* **GitHub:** Alojamiento del código y despliegue mediante GitHub Pages.

---

## 📂 Contenido del Repositorio
El repositorio está organizado en tres archivos de ejemplo que evolucionan en complejidad:

1.  **`example1.html`**: Introducción básica al concepto.
2.  **`example2.html`**: Aplicación intermedia y caso de uso real.
3.  **`example3.html`**: Uso de grid-template-areas y grid-area.
